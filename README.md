# ZomBiz-Survival-Guide

## Configuration

### Frontend

#### Install
```
cd frontend/zombie
npm install
```
#### Run
```
cd frontend/zombie
npm start
```

### Backend

#### Enviroment Variables
- STRIPE_PUBLISHABLE_KEY
- STRIPE_SECRET_KEY
- STRIPE_ENDPOINT_SECRET
- FLASK_SECRET_KEY

#### Install
```
cd backend/zombie
pip install -r requirements.txt
```

#### Run
```
cd backend/zombie
npm install
npm start
```
